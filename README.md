# Algorithms-for-combinatorial-optimization-of-parcel-delivery-locker-placement-problems

Θέμα: Αλγόριθμοι Συνδυαστικής Βελτιστοποίησης για Προβλήματα Χωροθέτησης Θυρίδων Παράδοσης Δεμάτων<br>
Συγγραφέας: Ειρήνη Μαρκαντώνη<br>
Επιβλέπων Καθηγητής: Νικόλαος Πλόσκας<br>
Τμήμα: Ηλεκτρολόγων Μηχανικών και Μηχανικών Υπολογιστών<br>
Ίδρυμα: Πανεπιστήμιο Δυτικής Μακεδονίας<br>
Ημερομηνία: Φεβρουάριος 2026<br>

Το παρόν repository περιλαμβάνει κώδικα που αναπτύχθηκε στο πλαίσιο της διπλωματικής εργασίας με θέμα: «Αλγόριθμοι συνδυαστικής βελτιστοποίησης για προβλήματα χωροθέτησης θυρίδων παράδοσης δεμάτων». Στόχος της εργασίας είναι ο προσδιορισμός των βέλτιστων θέσεων και τύπων θυρίδων (parcel lockers) σε αστικό περιβάλλον, με σκοπό την ελαχιστοποίηση της συνολικής απόστασης που διανύουν οι πελάτες για την παραλαβή των δεμάτων τους. Το πρόβλημα διατυπώνεται ως ένα ετερογενές πρόβλημα χωροθέτησης εγκαταστάσεων, το οποίο αποτελεί επέκταση του προβλήματος p-median με περιορισμούς χωρητικότητας, κόστους, προσβασιμότητας και περιβαλλοντικής επίδρασης. Η μοντελοποίηση λαμβάνει υπόψη πολλαπλούς τύπους θυρίδων, διαφορετικά μέσα μεταφοράς και στοχαστική κατανομή ζήτησης, ώστε να προσεγγίζει ρεαλιστικά σενάρια αστικής εφοδιαστικής. Η επίλυση του προβλήματος βασίστηκε σε μεθόδους ακέραιου γραμμικού προγραμματισμού (MILP), σε ανάπτυξη custom αλγορίθμων Branch & Bound, καθώς και σε ευρετικές και μεθευρετικές τεχνικές για την αντιμετώπιση της υπολογιστικής πολυπλοκότητας.<br>

Το αποθετήριο περιλαμβάνει:<br>

- Μοντέλα ακέραιου γραμμικού προγραμματισμού (MILP) υλοποιημένα με χρήση του Gurobi,
- Custom υλοποιήσεις αλγορίθμων Branch & Bound (baseline και βελτιωμένη έκδοση),
- Ευρετικούς και μεθευρετικούς αλγορίθμους για την κατασκευή και βελτίωση λύσεων,
- Γεννήτρια συνθετικών προβλημάτων για τη δημιουργία instances διαφορετικού μεγέθους,
- Κώδικα για πειραματική αξιολόγηση και συγκριτική ανάλυση των μεθόδων.<br>
